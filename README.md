# JSE Energy Advisor — Landing Page

Landing page desenvolvida para a JSE Energy Advisor, consultoria especializada em Diagnóstico Energético Empresarial (DEE), com foco em identificar oportunidades de economia na conta de energia de empresas.

🔗 **Site no ar:** [jseenergia.com](https://jseenergia.com)

## Sobre o projeto

Site institucional completo, construído do zero em HTML, CSS e JavaScript puro — sem frameworks — como projeto prático de aprendizado em desenvolvimento back-end e deploy.

## Funcionalidades

- Design responsivo, com identidade visual própria (paleta azul petróleo, verde e dourado)
- Formulário de pré-qualificação com validação de campos
- Integração com WhatsApp, gerando mensagem personalizada conforme os dados preenchidos
- Rastreamento de conversão via Google Analytics 4
- Página de Política de Privacidade (LGPD)
- Deploy contínuo via GitHub Pages, com domínio próprio e HTTPS

## Stack

- HTML5 / CSS3 / JavaScript (vanilla)
- Google Analytics 4 (GA4)
- GitHub Pages + domínio customizado (GoDaddy)
